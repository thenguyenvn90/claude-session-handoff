# Support

## How to get help

- **Bug or unexpected output?** [Open an Issue](https://github.com/thenguyenvn90/claude-session-handoff/issues/new?template=bug_report.yml)
- **Feature request?** [Open a Feature Request](https://github.com/thenguyenvn90/claude-session-handoff/issues/new?template=feature_request.yml)
- **General question?** [Browse existing Issues](https://github.com/thenguyenvn90/claude-session-handoff/issues) — your question may already be answered

## Before opening an issue

1. Check that `SKILL.md` is in the correct skills directory (`~/.claude/skills/`)
2. Start a **new** Claude Code session after installing (skills load at session start)
3. Use an exact trigger phrase: `session handoff`, `wrap up session`, or `hand off`

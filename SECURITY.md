# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| main (latest) | Yes |

## Reporting a Vulnerability

This project is a Claude Code skill (Markdown-only, no executable code). Security issues are unlikely but may include:

- Instructions that could lead to unsafe behavior when followed by an AI agent
- Privacy issues in the handoff output format

**To report a vulnerability:**

1. Open a [GitHub Issue](https://github.com/thenguyenvn90/claude-session-handoff/issues) with the label `security`
2. Or email: thenguyen.ai.automation@gmail.com

**Response timeline:**
- Acknowledgement within 48 hours
- Assessment and fix within 7 days for confirmed issues

Please do not disclose security issues publicly before they have been addressed.
